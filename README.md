## Usage
```
usage: pygraphslam [-h] --input INPUT [--draw_last DRAW_LAST] [--save_gif]

Python Graph Slam

optional arguments:
  -h, --help            show this help message and exit
  --input INPUT         Input CLF File.
  --draw_last DRAW_LAST
                        Number of point clouds to draw.
  --save_gif            Save a GIF animation
```
